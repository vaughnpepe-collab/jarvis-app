# Frame — Creative Studio / Agency Template

An editorial, light-mode landing page for creative studios, agencies, designers,
and freelancers. Big Playfair Display type, a confident coral accent, and tasteful
motion. The stylistic opposite of a dark SaaS page — built to feel like a portfolio
that wins clients.

![Frame preview](preview.png)

## What's included

- ✅ **Single-file `index.html`** — no build step
- ✅ **9 sections** — nav, editorial hero, animated marquee, staggered work grid, numbered services, studio/about, big-quote testimonial, contact CTA, footer
- ✅ **Fully responsive** (375 / 768 / 1024 / 1440px)
- ✅ **Accessible** — keyboard focus, `prefers-reduced-motion` (disables marquee + reveals), SVG icons
- ✅ **Self-contained visuals** — project thumbnails are CSS duotone gradients, so there are no image dependencies to chase (swap in your own `<img>` easily)
- ✅ **Tasteful motion** — scroll reveals, hover scale on work cards, an infinite marquee

## Quick start

1. Open `index.html` in any browser.
2. Customize in the `tailwind.config` block in `<head>`:
   - **Accent** → `colors.coral`
   - **Neutrals** → `colors.ivory` / `cream` / `ink`
   - **Fonts** → `fontFamily` (display = Playfair Display, body = Inter, labels = JetBrains Mono)
3. Replace work thumbnails: swap the `<div class="work-thumb" style="background:…">` for an `<img>`.
