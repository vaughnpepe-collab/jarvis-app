# NOXEN — Streetwear Label

Coined brand name (from *nox*, Latin "night"). Concept: after-dark / nocturnal streetwear.
First drop spans four aesthetics under one label.

## Designs (print-ready, 4500×5400 px @ 300dpi, transparent PNG — for black garments)
- `print/noxen-luxury.png` — minimal luxury (Gloock serif wordmark + "Nocturnal Division")
- `print/noxen-bold.png` — bold graphic ("NOXEN" heavy + "AFTER DARK" outline)
- `print/noxen-y2k.png` — Y2K / techno (pixel + chromatic glitch + system text)
- `print/noxen-grunge.png` — grunge / scrawl (distressed wordmark + handwritten "out past dark")
