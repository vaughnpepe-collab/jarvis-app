# The Macro-Structure Method — One-Page Cheat Sheet

> Print me. Educational only — **not financial advice.** Demo-test first.

## The whole method in one line
**Macro tide → liquidity sweep → structure shift → asymmetric trade, managed ruthlessly.**

---

## 1. BIAS (do this first — module 02)
- **Story:** one sentence. *"____ → bias on ____ is ____."* Can't say it = no bias.
- **Chart:** H4 close vs. **50-EMA** (rising = longs / falling = shorts) + HH-HL or LH-LL.
- Story and chart **agree** → trade that direction only. **Conflict → stand aside.**

## 2. ENTRY CHECKLIST (module 04) — ALL or no trade
```
[ ] H4 bias aligned (long-only OR short-only)
[ ] In session (London/NY, server time) + spread OK
[ ] LIQUIDITY SWEEP  : wick beyond a swing high/low, CLOSES back through
[ ] STRUCTURE SHIFT  : closes beyond the opposite swing (BOS / CHoCH)
[ ] Stop fits beyond the sweep (+ ATR buffer)
[ ] Reward:Risk to next liquidity ≥ 3 : 1
[ ] No high-impact news in the next ~30 min
```
One box unticked = **NO trade.** Forcing it = gambling.

## 3. THE TRADE PICTURE (long; mirror for short)
```
                              ┌─ TP: next swing high (liquidity)   reward 3R+
   swing high ──────┐         │
            ╱╲      │  ← close ABOVE swing high = BOS → ENTER
 ──────────╱──╲─────┼──────────────────────
  swing low╱    ╲   │
            ╲╱ sweep & reclaim
             └─ SL: below sweep low − ATR buffer   risk 1R
```

## 4. RISK (module 05 — the real edge)
- **Risk % per trade:** start **0.25–0.5%**. Fixed and small.
- **Size = (equity × risk%) ÷ loss-per-lot at your stop.** *Never a fixed lot.*
- Tight stop → bigger size; wide stop → smaller size; **dollar risk identical.**
- **Conviction up-size** only when everything aligns — capped at ≤ **2%**.
- Correlated pairs = one bet. Set a **daily loss limit**; hit it → stop for the day.

## 5. MANAGE (module 06)
- **+1R → stop to break-even** (trade now free).
- After +1R → **trail by ATR** (ride it) *or* take fixed 3R target.
- **Never** widen a stop or add to a loser.

## 6. EXPECTANCY (why 3:1 wins)
| Win rate | R:R | Edge/trade |
|---|---|---|
| 50% | 3:1 | +1.0R |
| 40% | 3:1 | +0.6R |
| 35% | 3:1 | +0.4R |

*It's not the win rate — it's win size vs. loss size.*
